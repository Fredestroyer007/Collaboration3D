Le dossier « Archive » contient le modèle Tigre que nous utilisions au début de la distribution à l'hôpital Sacré-Coeur

The "Jaguar" folder contains the Jaguar models we are currently producing! This is the model you should print!
The “Archive” folder, on the other hand, contains the Tiger model that we used at the start of distribution at the 
Sacré-Coeur hospital of Montreal. The Tiger model is now obsolete. You shouldn't print it!
